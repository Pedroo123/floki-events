# floki-events

A simple experiment with FLOCI and event driven approach.

Also, created thinking on Clean Arch, all made with .net 8.0


We have the following services configured:

1. A producer
2. SNS and SQS
3. A consumer
4. Bucket
5. Lambda
6. Dynamo to save data just because
